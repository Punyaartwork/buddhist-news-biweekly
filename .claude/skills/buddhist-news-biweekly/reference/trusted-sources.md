# Trusted Sources — buddhist-news-biweekly

Whitelist of domains the skill is allowed to scan and cite. Only these domains may appear as `source` in a primary pick or runner-up.

Review quarterly. When dropping a dead source, leave a one-line comment with the date and reason. When adding, include region + topical scope.

## Source policy

- A candidate may only be promoted to Primary Pick if it can be verified against at least one Tier A, B, or C domain.
- Tier D may only be used as a secondary citation inside a brief, never as the lead source.
- If a source appears repeatedly with biased framing or fact errors, file an issue and consider demotion to Tier D or removal.
